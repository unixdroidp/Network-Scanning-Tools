## Usage

1. Ensure Python is installed with tkinter (usually included).
2. Run the script:

   ```
   python network_tools.py
   ```

3. A professional GUI window will appear (500x600). Click on the button for the tool you want to use.
4. For browser history, a larger scrollable window (800x600) will open showing the history with timestamps.

## Requirements

- Python 3.x with tkinter
- Windows OS (for firewall check and traceroute)
- Google Chrome installed (for browser history)

No external dependencies required.
